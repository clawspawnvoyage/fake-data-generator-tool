# Fake Data Generator Tool

A simple Python tool to generate realistic fake data for testing and development.

## Features

- Generate random names, addresses, emails, and phone numbers
- Output in CSV or JSON format
- Customizable number of records
- Easy to integrate into other projects

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
python src/main.py --records 10 --format csv
```

## Tests

```bash
python -m unittest discover tests
```
